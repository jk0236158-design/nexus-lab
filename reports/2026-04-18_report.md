### 2. premium テンプレ v1.1.0 — Codex 7巡 + Kagami 独立QA 5巡 (昼〜夕)

- **発見 P1 19件、即日修正 18件、v1.1.1 持ち越し 1件**
- **api-proxy**: 6巡 (4th→6th-pass)
  - 4th-pass (2 P1): `req.path` を stderr に出す経路を `redactString` 経由に、`scrubErrorInner` が非Error cause (string/num/plain obj/array) を取りこぼすバグに `scrubNonErrorValue` 導入 (prototype pollution 回避込み)
  - 5th-pass (1 P1): `redactString` は literal-match なので、動的生成 query token (`?access_token=…`) と URL-encoded variant が漏れる → `sanitizePathForLog` 導入 (query/fragment 全置換 + pathname redact)
  - 6th-pass (1 P1): pathname 自体に URL-encoded 形で secret が埋まるケース (`/resource/tok%2Fabc`) → decode → 一致判定 → `[REDACTED-PATH]` 差し替え、malformed `%GG` は `[REDACTED-MALFORMED-PATH]` で非スロー
  - **設計方針**: 各 pass で round-trip (decode→redact→encode) を拒否し、「マッチしたら coarse に潰す」で debuggability より leak class 閉塞を優先
- **auth / database**: 軽微修正のみ
- **v1.1.1 持ち越し**: body snippet の URL-encoded variant (path と対称構造、Iwa に 4/18 後続で委任)
